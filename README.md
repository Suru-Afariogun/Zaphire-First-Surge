# Zaphire First Surge

   ![BubbleBlumanimeposter](Assets/Art-Graphic/Gallery%20posters%20and%20concept%20art/Bubble%20Blum%20anime%20poster.png)

**Zaphire First Surge** is a 2D action-platformer prototype created by **Suru Afariogun**.
It combines fast-paced combat, responsive movement, and dynamic boss encounters — all built in **Unity (C#)**.

This project serves as a technical and creative showcase of gameplay design, animation, and boss AI within a 2D environment.

---

## About the Project

**Zaphire First Surge** focuses on:

* Fluid 2D character movement and combat
* Memorable boss fights (e.g. Bubble Blum) with distinct patterns and telegraphs
* Jump-chase and slam-style boss behavior inspired by classic action games
* Local gameplay with controller and keyboard support

This game is being developed as a learning and portfolio project exploring Unity development, code structure, and visual design.

## Setup

To explore or open the project locally:

1. Clone or download the repository
2. Open the project folder in **Unity Hub**
3. Let Unity rebuild the necessary files automatically
4. Launch the project from the Unity Editor

> This repository contains only development-ready files; generated folders (Library, Temp, etc.) are ignored.

## Controls

Controls are defined in the project’s **PlayerControls** Input System Actions asset. **Keyboard** and **Gamepad** are supported.

| **Action**            | **Keyboard**               | **Gamepad**         |
|-----------------------|----------------------------|---------------------|
| **Move**              | W A S D / Arrow keys       | Left stick          |
| **Jump**              | Space / U                  | A (South)           |
| **Light Attack**      | M / I                      | X (West)            |
| **Heavy Attack**      | B / O                      | Y (North)           |
| **Dash**              | N / P                      | B (East)            |
| **Healing Core**      | Q / V (hold to charge)     | LB (Left shoulder)  |
| **Power Core**        | E / C (hold to charge)     | RB (Right shoulder) |
| **Combat Style Menu** | Tab                        | Start               |
| **Combat Style Left** | Z                          | LT (Left trigger)   |
| **Combat Style Right**| X                          | RT (Right trigger)  |
